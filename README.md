# Spoegwolf Daily Summary (starter)

Minimal Python scaffold that prints a WhatsApp-ready Afrikaans daily summary.
First data source: Plankton (placeholder endpoints).

## Quick start
```bash
python3 -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
python -m spoegwolf_daily.main
```

Edit `.env` with real Plankton credentials and `spoegwolf_daily/config.py` with show IDs.
